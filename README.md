# React Todo App

A simple **React Todo application** built with **Vite** to practice React fundamentals like state management, components, and event handling. Users can **add, delete, update tasks**, and manage daily to-dos efficiently.

---

## Live Demo
Try it online: [React Todo App Demo](https://sangardev.github.io/react-todo-app/)

---

## Features
- Add new tasks  
- Delete individual tasks  
- Update a task  
- Convert all tasks to uppercase (optional)  
- Interactive and responsive UI  
- Clean, simple design

---

## Tech Stack
- React  
- JavaScript (ES6+)  
- CSS3  
- Vite

---

## How to Run Locally in VS Code

Open project in VS Code

Go to File → Open Folder and select the project folder.

Open Terminal in VS Code

Go to View → Terminal or press Ctrl + `.

Install dependencies

npm install
Start the development server

npm run dev
Open in browser
